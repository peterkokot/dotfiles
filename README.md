# Peter's dotfiles

## About

This repository is configuration files to set up a system to my needs.

## Installation

### Prerequisites

* git
* vim
* ruby
* zsh
* [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
* wget

### Installation

```bash
git clone git://github.com/peterkokot/dotfiles.git --recursive ~/.dotfiles
.dotfiles/install
```
